# 我的创意：造页工坊 PageForge

## 一句话定位
像做 PPT 一样"造"网页——智能可视化 HTML 编辑器，让没有代码基础的人 10 分钟做出能用的网页。

## 核心闭环
自由画布拖拽组件 → 智能吸附对齐 + 规则推断转响应式 → 一键导出干净单文件 HTML（无平台锁定）

## 想解决什么问题
"做个网页"对普通人仍是高门槛任务。Webflow 虽强大但需理解 Flexbox、Grid、z-index 等 CSS 概念；Wix 零门槛但生成臃肿代码、强平台锁定；Framer 体验最好却不导出生产级代码。市面所有 HTML 编辑工具都在"设计自由度"和"响应式布局"之间二选一——没有任何工具真正做到"打开就能用、像拖 PPT 一样做网页、导出一份干净的 HTML 文件"。造页工坊用"智能吸附 + 规则推断"打破这个困局。

## 动机
我日常工作中常需快速做落地页、活动页、个人主页，找设计师排期要 1-2 周，自己学 Webflow 学习成本太高，用 Wix 又担心代码质量和锁定。调研发现 Webflow 2024 年营收突破 2 亿美元、Framer 2025 年站点数暴涨 400%、Wix 全球用户超 2.5 亿——需求真实且巨大，但"像 PPT 一样自由摆放 + 导出干净响应式 HTML"这个组合至今无人做到。借助 TRAE IDE 的 vibe coding 能力，普通人也能把这类复杂工具一步步造出来。

## 产品形态
Web 端智能可视化 HTML 编辑器。用户像做 PPT 一样在自由画布上拖拽组件（标题、图片、按钮、卡片、表单等）摆放布局，拖拽时智能吸附对齐、放下后规则推断自动转化为响应式 HTML/CSS，并支持一键导出干净的生产级单文件 HTML，可独立部署、无平台锁定。

## 差异化卖点（招牌功能，纯本地算法，0 API 成本）
1. 智能吸附对齐：拖拽时吸附到网格、对齐线、其他元素边缘（参考 WPS/PPT 智能对齐）
2. 规则推断响应式：根据元素 x/y 重叠关系推断"同行/同列/嵌套"，生成 Flex/Grid + flex-wrap 断点，破解"自由画布 vs 响应式"行业难题
3. 干净单文件 HTML 导出：AST → 语义化 HTML + 内联 CSS，零运行时依赖，单文件可直接双击打开，无平台锁定
4. PPT 式极简交互：自由画布 + 图层树 + 属性面板，学习成本接近 0

## 可选 AI Pro 功能（加分项，用参赛 Token 额度，成本可控时再启用，非核心）
- AI 整页生成：一句话描述需求，AI 生成完整可编辑页面骨架
- AI 文案/配色：根据页面主题自动生成标题文案与配色方案

- 商业价值：模板市场分佣 + 可选 AI Pro 订阅 + 企业版协作 + 一键托管增值
- 社会价值：降低"造网页"门槛，让小微企业、个体、学生都能拥有网络门面，缩小数字鸿沟，呼应大赛"人人都可以创造"理念

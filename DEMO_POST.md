### 0. 先和大家打个招呼吧 👋

- **你是谁：** 我是一名在校学生，平时喜欢借助 AI agent 工具独立开发项目。

- **你是怎么用 TRAE 把 Demo 做出来的：** 整个项目是我和 TRAE 一句句对话"聊"出来的。我的工作流很简单：用自然语言描述需求 → TRAE 生成代码 → 浏览器验证 → 发现 Bug → 文字描述让 TRAE 修复。听上去顺畅，实际上**几乎每个功能都经历了数十轮对话迭代**——拖拽吸附、图片裁切、HTML 导入导出，几乎没有一个是一次就做对的。每一轮都是"差一点、再改一点、再试一下"，直到效果达到预期。

  最让我头疼的是修**动效 Bug**——拖拽时预览跳了、旋转后拖拽飞了、裁切框按住手柄选框往外跳。这些 Bug 截图拍不到过程，对话框也不能发视频，我只能靠发送关键步骤截图搭配纯文字描述"在裁切窗口里面按住手柄的时候框会外弹一下扩大尺寸然后又迅速回到光标处"，让 agent 在完全看不到画面情况下理解问题。它经常"修错"——改了代码但 Bug 还在——因为它的理解和我的画面之间有天然 gap。我慢慢摸索出一些技巧：把 Bug 拆成一步一步描述，撰写尽量详细的复现步骤帮它建立画面感（"选区宽大于高时，拖动右手柄左移，期间成功出现吸附，但吸附之后一小段路程（鼠标未松）选区变成正方形等比缩小，即下边框也上移，此时仍不松手继续按住右手柄左移一段路程，吸附才消失，下边框回到正常位置不上移"），先让它读底层库源码再修。就这样一轮一轮对话，硬是把这些 Bug 都啃下来了。

  以前总觉得"一个人做不出这种东西"，但 agent 把门槛拉低了。现在我不敢想的是没有 TRAE 怎么做。

### 1. Demo 简介

- **是什么：** 造页工坊 PageForge 是一款浏览器内的可视化网页搭建工具（Web 应用），打开即用，无需安装。

- **面向谁：** 不会写代码却需要做网页的人——内容创作者、自媒体运营、小微企业主、学生、求职者。

- **主要功能：**

  🎨 **自由画布拖拽搭建** —— 像做 PPT 一样拖拽 14 种组件（标题、图片、按钮、卡片、表单、导航栏等）自由摆放。内置智能吸附对齐（边缘吸附、中心吸附、等间距吸附），拖拽时自动显示蓝色辅助线，放下即对齐。

  📥 **一键导入预设模板** —— 内置 9 套精心适配的模板（SaaS 落地页、作品集、简历等），一键加载到画布编辑。也支持粘贴 HTML 或上传文件导入。

  📤 **一键导出生产级 HTML** —— 导出为干净自包含的单文件 HTML，含响应式 CSS（桌面/平板/手机三层断点）、Google Fonts 国内镜像、零依赖交互运行时。可独立部署、无平台锁定。额外支持导出 PNG 和 PDF。

### 2. Demo 创作思路

- **灵感来源：** 身边很多同学和朋友需要做个人主页、作品集或活动页，但不会写代码。有时找我帮忙，我其实也不太精通。就想做一个工具，让他们自己拖一拖就能像做 PPT 一样搞定。

- **想解决的问题：** 把"找外包→沟通→设计→开发→部署"的流程压缩为"拖拽→导出"，让完全不会写代码的人在几分钟内做出能直接部署的网页。

- **为什么做这个方向：** 大赛主题是"人人都可以创造"。市面上很多工具要么学习曲线陡（Webflow 要理解 Flexbox/Grid），要么平台锁定（Wix 导不出代码），要么只托管不导出（Framer）。我想做一个**尊重用户产出物所有权**的工具——导出的 HTML 完全属于你，不被任何平台绑定。

  最让我头疼的是修**动效 Bug**——拖拽时预览跳了、旋转后拖拽飞了、裁切框按住手柄选框往外跳。这些 Bug 截图拍不到过程，对话框也不能发视频，我只能靠发送关键步骤截图搭配纯文字描述（如"在裁切窗口里面按住手柄的时候框会外弹一下扩大尺寸然后又迅速回到光标处"），让 agent 在几乎看不到操作步骤下理解问题。它经常"修错"——改了代码但 Bug 还在——因为它的理解和我的画面之间有天然 gap。我慢慢摸索出一些技巧：把 Bug 拆成一步一步描述，撰写尽量详细的复现步骤帮它建立画面感（如"选区宽大于高时，拖动右手柄左移，期间成功出现吸附，但吸附之后一小段路程选区变成正方形等比缩小，即下边框也上移，此时仍不松手继续按住右手柄左移一段路程，吸附才消失，下边框回到正常位置不上移"），先让它读底层库源码再修。就这样一轮一轮对话，硬是把这些 Bug 都啃下来了。

### 1. Demo 简介

- **是什么：** 造页工坊 PageForge 是一款浏览器内的可视化网页搭建工具（Web 应用），打开即用，无需安装。

- **面向谁：** 不会写代码却需要做网页的人——内容创作者、自媒体运营、小微企业主、学生、求职者。也面向**用 AI 生成网页后需要可视化精修的人**——把 DeepSeek/GLM/MiniMax写的 HTML 粘贴进来，秒变可视化编辑器。

- **主要功能：**

  - 🧩 **14 种组件 + 自由画布** —— 标题、正文、图片、按钮、卡片、容器、分隔线、图标、视频、输入框、导航栏、网格布局、表单、嵌入（iframe）。所有组件可在画布上自由拖拽摆放、缩放、旋转、镜像。内置**智能吸附对齐**（边缘 / 中心 / 等间距三套），拖拽时实时显示蓝色辅助线。画布支持**标尺**（按 R 切换光标定位线）、**缩放控制**（50%/100%/200% + 滚轮缩放）、**空格手型工具**、**4 种预设画布尺寸**（1280×720 / 1200×800 / 1024×768 / 1920×1080 + 自定义）。

  - ✂️ **图片处理** —— 双击图片打开裁切模态框，支持**矩形 / 圆形 / 圆角**三种形状切换，**8 向手柄**，叠加三套吸附系统：**正方形磁吸**（2.67x 滞后比，绿框指示）、**自由拉伸原比例磁吸**（基于原图自然尺寸）、**居中 / 边缘吸附**。图片支持**旋转**（-180°~180° 滑块）、**水平 / 垂直翻转**、**本地上传**或 URL 两种方式，裁切后可随时**重新裁切**。

  - 📐 **图层树 + 多选对齐** —— 图层树支持嵌套树形展示、拖拽排序、显示/隐藏切换、折叠展开、上下移/删除。多选时自动弹出**对齐工具栏**（六向对齐 + 水平/垂直等距分布，等距分布后显示间距标注）。画布**右键菜单**（上移一层 / 下移一层 / 复制 / 粘贴）。

  - 🧩 **两种编辑模式，智能切换** —— **自由画布**（像 PPT 一样自由拖拽，绝对定位，所见即所得）和 **精修模式**（iframe 渲染，100% 保留原 HTML 布局，适合复杂页面）。导入 HTML 时通过 **12 个信号智能检测**（flex/grid 数量、嵌套深度、@media、伪元素、calc/vh/vw 等），自动推荐最佳模式，用户也可手动切换。**特别适合精修 AI 生成的网页**——把 DeepSeek/GLM/MiniMax写的 HTML 粘贴进来，直接在页面上双击改文字、拖拽调尺寸、换配色，不用对着代码找标签。精修模式下支持**双击内联编辑文本**、**8 向缩放手柄**、**样式编辑器**（颜色/字号/字重/对齐/内边距/圆角）、**属性编辑器**（src/href/alt/title）、**DOM 面包屑导航**（点击跳转任意祖先元素）、**独立的撤销/重做系统**（debounced 事务合并，最多 80 条历史）。

  - 📚 **18 套模板 + HTML 导入** —— 9 套**预设模板**（专业简历 / SaaS 落地页 / Claude 编辑风 / GitHub 暗色 / 极简瑞士 / 杂志风等）精心适配到像素级；9 套**开源模板**（Start Bootstrap 系列，含 Agency、Freelancer、Creative 等）。支持粘贴 HTML 或上传 `.html` 文件，导入时自动弹出智能推荐弹窗，用户可选择自由画布或精修模式，并支持**重新生成**从已导入模板还原回编辑状态。

  - 🎨 **属性面板 + 预设样式** —— 字体编辑器（21 种字体族 + 粗体/斜体/下划线/删除线 + 字号/粗细/颜色）、外观（背景色/圆角/边框预设）。**按钮 8 种预设**（主要/次要/成功/危险/轮廓/圆角/大号/小号）、**卡片 6 种预设**（默认/浮起/边框/扁平/深色/带色）。**图标选择器**（SVG 矢量 + emoji 备选网格）。**格式刷**（复制一个元素的样式应用到另一个）。

  - 🎬 **交互动画（导出后仍保留）** —— 链接（`_self` / `_blank`）、点击动作（跳转 URL / 滚动锚点 / 切换显隐 / 提交表单）、悬停效果（缩放 / 阴影 / 变色 / 发光，过渡时长可调）、入场动画（淡入/滑入/缩放/弹跳等 7 种，`load` 或 `scroll` 触发，触发阈值可调）。所有交互以 `data-pf-*` 属性序列化进 HTML，由**零依赖交互运行时**驱动。

  - 📤 **三种导出格式** —— 导出 **HTML**（干净自包含单文件，含响应式 CSS 三层断点、Google Fonts 国内镜像加速、交互运行时，可独立部署）、**PNG**、**PDF**。导出的 HTML 完全属于你，无水印无平台锁定。

  - ⌨️ **全键盘操作** —— Ctrl+Z / Ctrl+Shift+Z 撤销重做、Ctrl+C / V / D 复制粘贴重复、Delete 删除、方向键微移（Shift+方向键=10px 大步）、Space 手型工具、R 切换标尺光标线。编辑区智能判断，输入框中不误触快捷键。

### 2. Demo 创作思路

- **灵感来源：** 身边很多同学和朋友需要做个人主页、作品集或活动页，但不会写代码。有时找我帮忙，我其实也不太精通。就想做一个工具，让他们自己拖一拖就能像做 PPT 一样搞定。

- **想解决的问题：** 把"找外包→沟通→设计→开发→部署"的流程压缩为"拖拽→导出"——让完全不会写代码的人在几分钟内做出能直接部署的网页。还有一个越来越常见的场景：**很多人用 AI 生成 HTML 页面，但 AI 给的代码没法直观修改**——改颜色要找到对应的 CSS 类名、改文字要在代码里翻标签、调尺寸要猜哪个 div 是哪个。PageForge 把 AI 生成的 HTML 粘贴进来，秒变可视化编辑器，双击改文字、拖拽调尺寸，所见即所得。

- **为什么做这个方向：** 大赛主题是"人人都可以创造"。市面上很多工具要么学习曲线陡（Webflow 要理解 Flexbox/Grid），要么平台锁定（Wix 导不出代码），要么只托管不导出（Framer）。更关键的是——**"自由"和"保真"是冲突的**：绝对定位画布自由度高但复杂布局会错位，iframe 渲染保真但受原结构约束。我的解决方案是**两条路径并存**，让用户自己选，而不是替用户决定。导出的 HTML 完全属于你，不被任何平台绑定。

### 5. 踩坑与心得

**坑一：拖拽预览"到处飞"——一个 Bug 四个根因**

从组件库拖拽元素到画布，松手后落点相对预览偏移（上下左右都偏）。表面看是一个 Bug，拆开发现是四个独立根因叠加：modifier 返回了绝对坐标而非 delta、落点用 overlay 中心计算但节点按左上角定位、吸附偏移在重置后才被读取、预览 div 的 `position:absolute` 与落点样式不一致。让 TRAE 逐层分析 dnd-kit 源码中的 `PositionedOverlay` 定位机制，四刀全修。

**坑二：旋转图片拖拽 50% 缩放时"飞"走**

100% 缩放时正常，50% 缩放时预览偏移。深入 dnd-kit 源码发现 `activeNodeRect` 使用 `getTransformAgnosticClientRect`（忽略 CSS transform），导致旋转补偿逻辑是死代码。真正根因是画布 `transformOrigin: 'top left'` 与 DragOverlay `transformOrigin: 'center center'` 不一致，zoom ≠ 100% 时产生 `(w*(1-zoom)/2, h*(1-zoom)/2)` 的视觉偏移。

**心得：和 agent 协作修动效 Bug 的方法**

动效 Bug 截图拍不到过程、不能发视频，纯靠文字描述。我总结出三个有效方法：

- **拆成步骤**：不描述"拖拽有问题"，而是"第一步 mousedown ✓，第二步移动 ✓，第三步松手 ✗ 停在鼠标右下 80px"
- **用类比建立画面感**："预览就像从元素左上角弹射出去，距离和元素大小有关"
- **先让 agent 读源码**：理解底层库机制后再修，避免 agent 在错误假设上反复修改

### 1. Demo 简介

- **是什么：** 造页工坊 PageForge 是一款浏览器内的可视化网页搭建工具（Web 应用），打开即用，无需安装。

- **面向谁：** 不会写代码却需要做网页的人——内容创作者、自媒体运营、小微企业主、学生、求职者。也面向**用 AI 生成网页后需要可视化精修的人**——把 DeepSeek/GLM/MiniMax写的 HTML 粘贴进来，秒变可视化编辑器。

- **主要功能：**

  - 🧩 **14 种组件 + 自由画布** —— 标题、正文、图片、按钮、卡片、容器、分隔线、图标、视频、输入框、导航栏、网格布局、表单、嵌入（iframe）。所有组件可在画布上自由拖拽摆放、缩放、旋转、镜像。内置**智能吸附对齐**（边缘 / 中心 / 等间距三套），拖拽时实时显示蓝色辅助线。画布支持**标尺**（按 R 切换光标定位线）、**缩放控制**（50%/100%/200% + 滚轮缩放）、**空格手型工具**、**4 种预设画布尺寸**（1280×720 / 1200×800 / 1024×768 / 1920×1080 + 自定义）。

  - ✂️ **图片处理** —— 双击图片打开裁切模态框，支持**矩形 / 圆形 / 圆角**三种形状切换，**8 向手柄**，叠加三套吸附系统：**正方形磁吸**（2.67x 滞后比，绿框指示）、**自由拉伸原比例磁吸**（基于原图自然尺寸）、**居中 / 边缘吸附**。图片支持**旋转**（-180°~180° 滑块）、**水平 / 垂直翻转**、**本地上传**或 URL 两种方式，裁切后可随时**重新裁切**。

  - 📐 **图层树 + 多选对齐** —— 图层树支持嵌套树形展示、拖拽排序、显示/隐藏切换、折叠展开、上下移/删除。多选时自动弹出**对齐工具栏**（六向对齐 + 水平/垂直等距分布，等距分布后显示间距标注）。画布**右键菜单**（上移一层 / 下移一层 / 复制 / 粘贴）。

  - 🧩 **两种编辑模式，智能切换** —— **自由画布**（像 PPT 一样自由拖拽，绝对定位，所见即所得）和 **精修模式**（iframe 渲染，100% 保留原 HTML 布局，适合复杂页面）。导入 HTML 时通过 **12 个信号智能检测**（flex/grid 数量、嵌套深度、@media、伪元素、calc/vh/vw 等），自动推荐最佳模式，用户也可手动切换。**特别适合精修 AI 生成的网页**——把 DeepSeek/GLM/MiniMax写的 HTML 粘贴进来，直接在页面上双击改文字、拖拽调尺寸、换配色，不用对着代码找标签。精修模式下支持**双击内联编辑文本**、**8 向缩放手柄**、**样式编辑器**（颜色/字号/字重/对齐/内边距/圆角）、**属性编辑器**（src/href/alt/title）、**DOM 面包屑导航**（点击跳转任意祖先元素）、**独立的撤销/重做系统**（debounced 事务合并，最多 80 条历史）。

  - 📚 **18 套模板 + HTML 导入** —— 9 套**预设模板**（专业简历 / SaaS 落地页 / Claude 编辑风 / GitHub 暗色 / 极简瑞士 / 杂志风等）精心适配到像素级；9 套**开源模板**（Start Bootstrap 系列，含 Agency、Freelancer、Creative 等）。支持粘贴 HTML 或上传 `.html` 文件，导入时自动弹出智能推荐弹窗，用户可选择自由画布或精修模式，并支持**重新生成**从已导入模板还原回编辑状态。

  - 🎨 **属性面板 + 预设样式** —— 字体编辑器（21 种字体族 + 粗体/斜体/下划线/删除线 + 字号/粗细/颜色）、外观（背景色/圆角/边框预设）。**按钮 8 种预设**（主要/次要/成功/危险/轮廓/圆角/大号/小号）、**卡片 6 种预设**（默认/浮起/边框/扁平/深色/带色）。**图标选择器**（SVG 矢量 + emoji 备选网格）。**格式刷**（复制一个元素的样式应用到另一个）。

  - 🎬 **交互动画（导出后仍保留）** —— 链接（`_self` / `_blank`）、点击动作（跳转 URL / 滚动锚点 / 切换显隐 / 提交表单）、悬停效果（缩放 / 阴影 / 变色 / 发光，过渡时长可调）、入场动画（淡入/滑入/缩放/弹跳等 7 种，`load` 或 `scroll` 触发，触发阈值可调）。所有交互以 `data-pf-*` 属性序列化进 HTML，由**零依赖交互运行时**驱动。

  - 📤 **三种导出格式** —— 导出 **HTML**（干净自包含单文件，含响应式 CSS 三层断点、Google Fonts 国内镜像加速、交互运行时，可独立部署）、**PNG**、**PDF**。导出的 HTML 完全属于你，无水印无平台锁定。

  - ⌨️ **全键盘操作** —— Ctrl+Z / Ctrl+Shift+Z 撤销重做、Ctrl+C / V / D 复制粘贴重复、Delete 删除、方向键微移（Shift+方向键=10px 大步）、Space 手型工具、R 切换标尺光标线。编辑区智能判断，输入框中不误触快捷键。

### 2. Demo 创作思路

- **灵感来源：** 身边很多同学和朋友需要做个人主页、作品集或活动页，但不会写代码。有时找我帮忙，我其实也不太精通。就想做一个工具，让他们自己拖一拖就能像做 PPT 一样搞定。

- **想解决的问题：** 把"找外包→沟通→设计→开发→部署"的流程压缩为"拖拽→导出"——让完全不会写代码的人在几分钟内做出能直接部署的网页。还有一个越来越常见的场景：**很多人用 AI 生成 HTML 页面，但 AI 给的代码没法直观修改**——改颜色要找到对应的 CSS 类名、改文字要在代码里翻标签、调尺寸要猜哪个 div 是哪个。PageForge 把 AI 生成的 HTML 粘贴进来，秒变可视化编辑器，双击改文字、拖拽调尺寸，所见即所得。

- **为什么做这个方向：** 大赛主题是"人人都可以创造"。市面上很多工具要么学习曲线陡（Webflow 要理解 Flexbox/Grid），要么平台锁定（Wix 导不出代码），要么只托管不导出（Framer）。更关键的是——**"自由"和"保真"是冲突的**：绝对定位画布自由度高但复杂布局会错位，iframe 渲染保真但受原结构约束。我的解决方案是**两条路径并存**，让用户自己选，而不是替用户决定。导出的 HTML 完全属于你，不被任何平台绑定。

### 5. 踩坑与心得

**坑一：拖拽预览"到处飞"——一个 Bug 四个根因**

从组件库拖拽元素到画布，松手后落点相对预览偏移（上下左右都偏）。表面看是一个 Bug，拆开发现是四个独立根因叠加：modifier 返回了绝对坐标而非 delta、落点用 overlay 中心计算但节点按左上角定位、吸附偏移在重置后才被读取、预览 div 的 `position:absolute` 与落点样式不一致。让 TRAE 逐层分析 dnd-kit 源码中的 `PositionedOverlay` 定位机制，四刀全修。

**坑二：旋转图片拖拽 50% 缩放时"飞"走**

100% 缩放时正常，50% 缩放时预览偏移。深入 dnd-kit 源码发现 `activeNodeRect` 使用 `getTransformAgnosticClientRect`（忽略 CSS transform），导致旋转补偿逻辑是死代码。真正根因是画布 `transformOrigin: 'top left'` 与 DragOverlay `transformOrigin: 'center center'` 不一致，zoom ≠ 100% 时产生 `(w*(1-zoom)/2, h*(1-zoom)/2)` 的视觉偏移。

**心得：和 agent 协作修动效 Bug 的方法**

动效 Bug 截图拍不到过程、不能发视频，纯靠文字补充描述。我总结出三个有效方法：

- **拆成步骤**：不描述"拖拽有问题"，而是"第一步 mousedown ✓，第二步移动 ✓，第三步松手 ✗ 停在鼠标右下约 80px"
- **用类比建立画面感**："预览就像从元素左上角弹射出去，距离和元素大小有关"
- **先让 agent 读源码**：理解底层库机制后再修，避免 agent 在错误假设上反复修改

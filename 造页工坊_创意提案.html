01 / 创意介绍

一个被遗忘了二十年的梦

让普通人也能像做 PPT 一样做网页——这是 Dreamweaver 时代就有的需求，却至今没有工具真正做到。

想解决什么问题

"做个网页"对普通人仍是高门槛。Webflow 要懂 CSS，Wix 代码臃肿，Framer 不导出代码。市面所有工具都在"设计自由度"和"响应式"之间二选一。

行业级困局

为什么会想到做这个

自己常需快速做落地页，找设计师要 1–2 周，学 Webflow 太陡，用 Wix 担心锁定。调研发现 Webflow 年营收 2 亿美元、Framer 站点 +400%、Wix 2.5 亿用户——需求真实巨大。

需求验证

大概是什么产品

Web 端智能可视化编辑器。用户像做 PPT 一样拖拽组件摆放，智能吸附对齐、规则推断实时转成响应式 HTML/CSS，一键导出干净单文件 HTML，可独立部署、无锁定。

06 / 智能与 AI 能力

招牌靠算法，AI 是加分项

招牌功能用纯本地算法实现，0 API 成本、0 延迟；AI 作为可选 Pro 功能，用参赛 Token 额度，成本可控时再启用。
智能吸附对齐招牌拖拽时吸附到网格、对齐线、其他元素边缘，参考 WPS/PPT 智能对齐，纯本地算法。
规则推断响应式招牌根据元素 x/y 重叠关系推断同行/同列/嵌套，生成 Flex/Grid + flex-wrap 断点，破解"自由画布 vs 响应式"难题。
干净 HTML 序列化招牌AST → 语义化 HTML + 内联 CSS，零运行时依赖，单文件可直接双击打开。
AI 整页生成可选 Pro一句话描述需求，AI 生成完整可编辑页面骨架，用参赛 Token 额度。
AI 文案/配色可选 Pro根据页面主题自动生成标题文案与配色方案，成本可控时启用。
创作全流程 vibe coding过程从创意到 Demo 全程用 TRAE IDE 对话式开发，契合大赛"用 AI 把创意造成作品"。

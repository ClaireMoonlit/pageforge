# 【学习工作赛道】造页工坊 PageForge —— 零代码可视化网页搭建工具

**【标签】** 学习工作

---

## 1. 创意名称 + 创意介绍

**创意名称**：造页工坊 PageForge

**想解决什么问题**：
很多非技术人员——小商家、自由职业者、学生——需要一个简单的网页来展示自己的产品或作品集，但现有工具要么太复杂（Webflow、WordPress），要么模板太死板（Wix、Canva），要么需要学代码。他们想要的是：**像做 PPT 一样做网页**——拖拽就能搭，搭完就能用。

**为什么会想到做这个**：
我身边有很多朋友想给自己做个人主页、活动落地页，但每次打开网页编辑器就被技术术语劝退。正好我在用 TRAE IDE 学习前端开发，就想：既然 AI 能帮我写代码，那我能不能用 AI 帮别人"不用写代码也能做网页"？

**大概是什么产品**：
一个浏览器内的可视化网页搭建网站（Web App），用户拖拽组件到画布、用属性面板调样式、一键导出可在任意服务器部署的 HTML 文件。

---

## 2. 目标用户及痛点

**面向哪些用户**：
- 小商家 / 个体户：需要产品展示页、活动落地页
- 自由职业者 / 设计师：需要作品集 / 个人主页
- 学生社团：需要活动宣传页
- 运营人员：需要营销落地页做 A/B 测试

**在什么场景下使用**：
老板说"明天活动要个报名页"——运营打开 PageForge，导入一个模板，改改文字和图片，10 分钟导出 HTML 发给开发部署。不需要排期等前端。

**当前痛点**：
- 找外包做网页 → 贵、慢、沟通成本高
- 用现成模板平台 → 改不动、导出受限、长一样
- 自己学前端 → 学习曲线陡、劝退率高
- 用 AI 直接生成代码 → 不可控、改不了细节

---

## 3. 价值与意义

**效率提升**：
传统方式做一张落地页：需求沟通 → 设计稿 → 前端开发 → 联调 → 上线，至少 2-3 天。PageForge 把整个过程压缩到 10-30 分钟，且非技术人员可独立完成。

**创作门槛降低**：
让"不会写代码的人"也能拥有自己定制的网页。不是模板填空，而是真正的自由画布拖拽 + 属性精修 + 所见即所得。

**技术价值**：
- 整个项目由 **TRAE IDE + AI 对话** 驱动开发完成，是"人机协作编程"的典型案例
- 内置 9 套免费模板（Start Bootstrap 系列），一键导入即可编辑
- 支持交互功能（点击隐藏/显示、链接跳转、入场动画、悬停效果）
- 导出为纯 HTML 单文件，零依赖，可直接部署到任何服务器
- 响应式导出，手机端自适应

**自举验证**：
PageForge 本身就是一个网页应用，而它的导出物也是网页——**用 TRAE 造的工具，造出来的网页还能继续参赛**。这是对"AI 创造力"最直接的自证。
